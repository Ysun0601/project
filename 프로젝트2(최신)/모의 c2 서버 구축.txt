모의 C2 서버 구축

1. EC2 인스턴스 생성 방법
목적
모의 C2 서버를 실행할 기반 환경을 설정.
AWS 클라우드에서 안전하게 테스트 환경을 운영.

단계별 구성
EC2 인스턴스 생성:
AWS Management Console에서 EC2 서비스로 이동.
"인스턴스 시작(Launch Instance)" 버튼 클릭.

AMI 선택:
Amazon Linux 2 또는 Ubuntu를 선택.

인스턴스 유형:
테스트 환경이므로 t2.micro 또는 t3.micro와 같은 소형 인스턴스 추천(프리 티어 가능).

키 페어(Key Pair):
새로운 키 페어를 생성하거나 기존 키 페어를 선택.

네트워크 설정:
VPC: 테스트를 위해 별도의 VPC나 서브넷에 배치.

보안 그룹:
SSH(포트 22) 허용(테스트 환경에서만 접근 가능하도록 제한).
모의 C2 서버에서 사용할 포트 허용(예: HTTP 80/443, 사용자 정의 포트 8080).

스토리지 구성:
기본 스토리지 크기를 8GB 이상으로 설정.

인스턴스 시작:
설정을 확인하고 "인스턴스 시작" 버튼 클릭.
인스턴스가 시작되면 퍼블릭 IP 주소를 복사해 준비.
